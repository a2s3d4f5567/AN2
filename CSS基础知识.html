CSS基础

CSS(Cascading Style Sheets)层级样式表，用以控制页面样式，实现页面中元素的表现形式、区块的布局、不同设备的适应等等，是 Web 标准中实现内容和表现相分离的唯一机制。

CSS使用方式

外部link 内部style 内嵌style
<link rel="stylesheet" href="style.css"> <style>...</style> <p style="color:red">

CSS语法

语法声明分成两类：at 规则 (at-rules) 和 CSS 规则集 (rule sets)。声明之间可由空白字符连接。

  • At 规则以"@"关键字开始，之后紧跟标志符。比如："@import"
  • 规则集由选择符和跟随其后的声明块组成。声明块以"{"开始，以"}"结束

选择符

分类

简单选择符 伪元素选择符 组合选择符 群组选择符
Type Selector Attribute Selector Class Selector h1, h2, h3

简单选择符

类型选择符（Type Selector），也叫做元素选择符：

h1 {color:red}

上述规则将使文档中所有的 h1 元素字体颜色为红色

属性选择符

属性选择符（Attribute Selector）将选定那些拥有匹配属性的元素

[type="text"] {border: 1px solid}

类选择符

类选择符（Class Selector）将选定那些拥有匹配类名的元素

.intro {font-weight: bold}

ID选择符

将选择 `ID` 属性等于指定值的元素。按照 W3C 标准，`ID` 属性在 `DOM` 中的值应该是唯一的。

#bar /*将选择所有具有 id="bar"属性值的元素*/ h1#bar /*将选择具有 id="bar"的 h1 元素*/

文本样式包括文本颜色、文本对齐方式、文本装饰等。

字体 斜体 粗体 大小 颜色 对齐 装饰 间距 阴影

背景样式

背景样式包括背景颜色、背景图片、背景位置、背景重复等。

背景颜色 背景图片 渐变背景
background-color: #f0f0f0; background-image: url('background.jpg'); background: linear-gradient(to right, red, blue);

盒模型

所有 HTML 元素，在页面的呈现过程中，都遵循 CSS 制定的盒模型（Box Model），盒模型是一个包含外边距（margin）、边框线（border）、内边距（padding）以及内容的矩形容器。

定位及布局

信息流

浏览器在呈现信息时会按照元素的类型进行处理，它将块元素从上到下显示（块元素与块元素之间另起一行），将行内元素按语言方向水平显示（如汉字、英语是从左到右，维吾尔语、阿拉伯语等有些语言是从右到左），行内元素直到到达容器边缘时才换行显示，这种显示元素的方式叫做页面的正常流。

display

设置元素的显示方式

元素如何显示，可以通过 `display` 属性进行指定。每一个元素都有一个默认的 `display` 属性，但这个属性值可以修改。块元素和行内元素可通过 `display` 属性改变它们的显示方式，比如在某些情景，编辑人员需要给行内元素添加高度，以改进元素的显示效果，行内元素是无法直接通过 `width` 属性指定宽度的，但可以通过 `display` 属性将其变为块元素，再为其添加高度。

margin: 10px;

position

属性用来设定对象的定位方式，是非常强大、灵活的改变元素信息流的方式之一。relative、absolute、fixed

Flexbox

Flex box布局模型的主要目的是提供更有效率的布局方式，尤其是当容器内元素的尺寸不固定的时候更能表现出它的优势。它能够自动识别子元素的尺寸，从而为盒装模型提供更高的灵活

flexbox 设定容器

flexbox 设定容器 任何一个容器都可以指定为Flexbox布局 #.flex-container { display: flex; }#
