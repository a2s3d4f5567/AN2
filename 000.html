    • HTML基础

    HTML是一种用于创建网页的标记语言。它使用标签来描述网页的结构和内容。

    HTML结构

    **HTML的结构** DYCTYPE html html head title 文档标题 title /head body /body /html

    内容

    HTML使用标签来标记网页的不同部分，如标题、段落、链接等。以下是一些常用的HTML标签：

    • <h1>到<h6>：用于标题
    • <p>：用于段落
    • <a>：用于链接
    • <ul>:用于列表
    • <div>:用于逻辑区块
    • <table>:用于表格
    • <form>:用于表单
    • <img>:用于插入图片

    HTML标签

    标签以“<”开始，以“>”结束

    • title：用于起始标签
    • p：用于结束标签

    HTML元素

    元素由起始标签、内容和结束标签构成，是HTML的基本单位

    • href：用于链接标签
    • src：用于图片标签

    HTML属性

    属性用于提供额外的信息给HTML元素，例如链接的目标URL或图片的来源。属性通常以“属性名=属性值”的形式出现。

    • href：用于链接标签
    • src：用于图片标签

    HTML注释

    HTML注释以“”结束。注释不会显示在浏览器中，但可以用于提供对代码的解释或说明。

    CSS基础

    CSS(Cascading Style Sheets)层级样式表，用以控制页面样式，实现页面中元素的表现形式、区块的布局、不同设备的适应等等，是 Web 标准中实现内容和表现相分离的唯一机制。

    CSS使用方式

    CSS语法

    语法声明分成两类：at 规则 (at-rules) 和 CSS 规则集 (rule sets)。声明之间可由空白字符连接。

      At 规则以“@”关键字开始，之后紧跟标志符。比如：“@import”
      规则集，也称规则，由选择符和跟随其后的声明块组成。声明块以“{”开始，以“}”结束，其中的声明以“;”分隔。声明由属性名称和属性值组成，属性名称和属性值之间用“:”连接

    CSS注释

    在 CSS 中，注释以“`/*`”开始，以“`*/`”结束，注释之内的内容会被浏览器忽略。注释不能嵌套使用。

    简答选择符

    类型选择符（Type Selector），也叫做元素选择符，该选择符能代表文档树中的所有指定的元素 #`h1 {color:red}`# 上述规则将使文档中所有的 `h1` 元素字体颜色为红色

    属性选择符

    属性选择符（Attribute Selector）将选定那些拥有匹配属性的元素

    类选择符

    类选择符（Class Selector）将选定那些拥有匹配类名的元素

    ID选择符

    ID选择符（ID Selector）将选定那些拥有匹配ID的元素

    背景

    • #background-color#:背景色
    • #background-image#:背景图片
    • #background: linear-gradient(#648880, #293f50)：渐变背景
    • #linear-gradient#:线性渐变

      • 盒模型

      所有 HTML 元素，在页面的呈现过程中，都遵循 CSS 制定的盒模型（Box Model），盒模型是一个包含外边距（margin）、边框线（border）、内边距（padding）以及内容的矩形容器。这些值可以单独设置，如top、right、 bottom、 left

      信息流、display以及position

      信息流

      浏览器在呈现信息时会按照元素的类型进行处理，它将块元素从上到下显示（块元素与块元素之间另起一行），将行内元素按语言方向水平显示（如汉字、英语是从左到右，维吾尔语、阿拉伯语等有些语言是从右到左），行内元素直到到达容器边缘时才换行显示，这种显示元素的方式叫做页面的正常流

      display

      display属性用于设置元素的显示方式，包括块级元素、行内元素、内联块元素等。

      • #display: block#:块级元素，会独占一行，默认宽度为父元素的100%，高度自适应内容。

        • position

        position 属性用来设定对象的定位方式，是非常强大、灵活的改变元素信息流的方式之一

    设定字体 创建斜体 应用粗体 字体大小 行间距 字体颜色 文本对齐 文本装饰 文字阴影

    flexbox

    Flex box布局模型的主要目的是提供更有效率的布局方式，尤其是当容器内元素的尺寸不固定的时候更能表现出它的优势。它能够自动识别子元素的尺寸，从而为盒装模型提供更高的灵活性
