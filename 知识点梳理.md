## 获取已有知识点

读取knowledge.md文档中的知识点

##  整理知识点

根据用户提供的信息，将其中的知识点，按照逻辑结构，用markdown语法进行整理。


### 知识点整理案例

<原文>
h1 {
  text-align: center;
  color: rgba(184, 108, 108, 0.856);
}
</原文>

提取的知识点如下：
```markdown
# 前端开发
## CSS
### CSS基本语法
#### 元素选择符
#### 文本属性
```
## 输出格式
```markdown
## 父知识点
### 子知识点1
#### 孙知识点1
……
### 子知识点2
……
```

## 写入文件

将整理的内容写入knowledge.md文件。
